## Proposta

A ideia do projeto é implementar a caminhada do robô formiga (com seis pernas). O hardware é composto por uma placa Toradex com compatibilidade Linux, duas MBEDS e 12 servomotores de aeromodelismo (2 em cada perna). E como ele foi decidido previamente, o foco do projeto ficou mais com a implementação do software no robô.

O trabalho então foi dividido em duas partes. A primeira pensando na lógica da movimentação para a caminhada, pensando mais nos momentos em que cada motor seria acionado. Já a segunda foi voltada mais para a comunicação entre as duas placas (e a divisão de processamento entre elas).
